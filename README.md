## How to use

- Step 1: Download all the files in this repository.
- Step 2: Open the file named `index.php` in your browser, the full url is given here: `http://localhost:3000/Real_Projects/Final_Project/index.php`
Note: if this url does not work, try to run a php server in Visual studio code. 
1. Download Visual Studio code, lnk given below:
[Download Visual Studio code](https://code.visualstudio.com/download)
2. Install the extension: `PHP Server` in Visual Studio code
3. Then right click.
4. Select the `Serve Project`. (Ensure you do this step in the index.php file only.)
- Step 4: Enjoy!
